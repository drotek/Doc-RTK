## Pinouts

### XL RTK GNSS (F9P)

<p align="center">
  <img src="./images/XL_F9_A.jpg?raw=true" alt="pinouts XL ZED-F9P drotek"/>
</p>

| Wire N° | Corresponding pinout |
|---------|----------------------|
| A1      | 5V IN                |
| A2      | SPI MOSI             |
| A3      | SPI MISO             |
| A4      | SPI CLK              |
| A5      | SPI CS               |
| A6      | GROUND               |

<p align="center">
  <img src="./images/XL_F9_BC.jpg?raw=true" alt="pinouts XL F9P ublox"/>
</p>

|---------|----------------------|
| B1      | 5V SPI               |
| B2      | I2C SCL              |
| B3      | HOST TX              |
| B4      | HOST RX              |
| B5      | I2C SDA              |
| B6      | GROUND               |
|         |                      |
| C1      | 5V IN                |
| C2      | RTCM3 RX             |
| C3      | RTCM3 TX             |
| C4      | RTK STAT             |
| C5      | GEOFENCE STAT        |
| C6      | GROUND               |
